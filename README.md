# 競プロ用テンプレート

GitHub Codespaces 上で、C++ / PyPy / Rust を使った競技プログラミング環境

## 使い方

1. このリポジトリを Fork
2. GitHub 上で Codespaces を起動
3. ターミナルで以下のコマンドを使って問題用ファイルを作成

## ファイル作成コマンド

```bash
newcpp abc001_a    # → cpp/abc001_a.cpp
newpy abc001_a     # → python/abc001_a.py
newrs abc001_a     # → rust/abc001_a.rs
```
